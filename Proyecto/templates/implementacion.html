Métricas de Desempeño

Se seleccionan y aplican métricas adecuadas según el tipo de problema. En modelos de regresión se usan métricas como MSE o R².

Validación

Se usa validación cruzada para comprobar qué tan bien se desempeña el modelo con distintos datos. En modelos de regresión, se aplican métricas como el MSE para ver el error promedio de las predicciones, y R² para saber qué tanto explican los datos al resultado.

Análisis de Riesgos

Se examinan posibles riesgos como el sobreajuste (modelo demasiado ajustado al entrenamiento), subajuste (modelo poco complejo), o sesgos en los datos, que podrían afectar la utilidad real del modelo.
