Entendimiento del Negocio y los Datos

Objetivos de Negocio

Traducir la necesidad de las autoridades de salud pública de Chía en metas concretas para el proyecto de machine learning: Predecir cuántos animales callejeros deberán ser vacunados por zona, para optimizar recursos y prevenir brotes de rabia.

Preguntas Clave

  • ¿Qué zonas tendrán mayor necesidad de vacunación en futuras jornadas?
  • ¿Cuántos animales callejeros deberíamos planear vacunar por zona?
  • ¿Qué variables (tipo de animal, zona, fechas) influyen más en esa demanda?

Evaluación de Datos

Se evaluará la calidad del censo actual de vacunación:

  • Cobertura geográfica (barrios o veredas)
  • Variables disponibles: especie, sexo, edad estimada, fecha, cantidad de vacunados
  • Posibles vacíos de datos, inconsistencias o duplicados
  • Se analizará si los datos tienen suficiente historial para detectar patrones por zona y por tiempo

Análisis de Viabilidad

Técnicamente viable:

Se cuenta con datos en formato estructurado, suficientes para análisisexploratorio y modelado.

Organizacionalmente viable:

El modelo puede ser útil para la alcaldía y entidades de salud pública local, permitiendo decisiones basadas en datos.

Limitaciones:

Puede haber zonas subregistradas o falta de fechas consistentes, esto requerirá preprocesamiento.
