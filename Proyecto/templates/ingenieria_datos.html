Fase de Ingeniería de Datos

Selección

Se identificó como fuente principal el dataset del censo de vacunación antirrábica de animales callejeros en Chía. Se seleccionaron columnas clave como año, mes, zona, tipo de jornada y totales vacunados.

Limpieza

Se eliminaron registros con valores faltantes críticos, y se imputaron NaN menores mediante el uso de valores promedio o por eliminación según el caso. También se estandarizaron nombres de columnas y se eliminaron caracteres especiales.

Transformación

Se crearon nuevas características como el total de animales vacunados por jornada. Se codificaron variables categóricas como "zona" y "mes" mediante codificación numérica. También se normalizaron los datos numéricos para estabilizar la varianza.

Documentación

Todas las transformaciones se registraron en el archivo `regresionLineal.py`, incluyendo pasos como la imputación, codificación, y generación de variables nuevas. Esto garantiza que el proceso sea totalmente reproducible y auditable en futuras versiones del modelo.
